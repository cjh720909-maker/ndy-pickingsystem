# ndy-pickingsystem

NDY 배차 및 피킹 요약 대시보드 시스템입니다.

## 주요 규칙
- **명칭 일관성**: 모든 내부 폴더, 깃허브 저장소, Vercel 프로젝트 명칭은 `ndy-pickingsystem`을 유지합니다.
- **데이터베이스 보호**: 배포 및 업데이트 시 기존 데이터베이스의 테이블 구조를 절대 수정하거나 삭제하지 않습니다.
- **보고서 언어**: 모든 보고서와 소통은 한국어로 진행합니다.

## 기술 스택
- **Frontend**: Vanilla Javascript, CSS
- **Backend**: Express, Prisma (MySQL)
- **Deployment**: Vercel
